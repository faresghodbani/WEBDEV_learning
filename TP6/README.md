# 🎮 INVERSEGAME — Le Gant de l'Infini

Bienvenue dans **INVERSEGAME**, un jeu interactif inspiré de l’univers Marvel où, pour une fois…

**VOUS êtes le méchant.**

Votre mission : récupérer les **6 Pierres d’Infinité**, survivre aux héros, et activer le **Gant de l’Infini** pour décider du destin de l’humanité.

---

# 📖 Histoire

Les héros ont échoué.

Les Avengers ont été dépassés.
Les Gardiens de la Galaxie ne peuvent plus protéger l’univers.

Les multivers sont instables, les pierres ont été dispersées…
et quelqu’un doit les retrouver.

Cette personne, c’est **vous**.

Mais les héros sont encore présents et tenteront de vous arrêter avant que vous ne réunissiez toutes les pierres.

Si vous réussissez…

Un simple **claquement de doigts** suffira pour faire disparaître l’humanité.

---

# 🎯 Objectif du jeu

Pour gagner la partie :

1️⃣ Trouver les **6 Pierres d’Infinité**
2️⃣ Survivre aux attaques des héros
3️⃣ Activer le **Gant de l’Infini**

⚠️ Vous devez **posséder les 6 pierres ET être encore en vie** pour gagner.

---

# ❤️ Système de vie

Vous commencez la partie avec :

**100 HP**

Certains ennemis peuvent vous attaquer.

| Ennemi                    | Effet  |
| ------------------------- | ------ |
| 💀 Avengers               | −50 HP |
| 🚀 Gardiens de la Galaxie | −50 HP |

Si votre vie tombe à **0 HP**, la partie est perdue.

---

# 💎 Inventaire des Pierres d’Infinité

Un **inventaire** affiche les pierres que vous avez collectées.

Objectif :

**6 / 6 pierres**

Chaque pierre trouvée s’ajoute à l’inventaire.

Lorsque les **6 pierres sont réunies**, le **Gant de l’Infini peut être activé**.

---

# 🧩 Mode principal : la grille

Dans le QG Marvel, vous trouverez une **grille de 8 cases**.

### Fonctionnement

* Cliquez sur une case pour la révéler
* Les **6 pierres** sont placées **aléatoirement**
* Certaines cases contiennent des ennemis

Chaque partie est différente.

---

# ⚠️ Les dangers

Certaines cases cachent des ennemis.

💀 **Avengers**
🚀 **Gardiens de la Galaxie**

Chaque rencontre inflige :

**−50 HP**

Deux mauvaises rencontres peuvent donc suffire à perdre la partie.

---

# 🔐 Le cadenas secret

Dans l’interface se trouve un **cadenas verrouillé**.

Pour l’ouvrir :

* Trouver le **code secret à 2 chiffres**
* Vous avez **5 tentatives maximum**

Si vous trouvez le bon code :

🔓 **Un accès secret se débloque.**

---

# 💡 Indice caché

Un **indice est dissimulé dans l’interface du jeu**.

Les joueurs attentifs pourront trouver le code du cadenas en observant bien l’environnement.

---

# 🌌 Le Bunker Secret

Un bouton discret permet d’accéder à une zone cachée :

**LE BUNKER**

Dans ce mode :

* Les **pierres tournent sur elles-mêmes**
* Vous devez toutes les récupérer
* Une fois réunies, un bouton apparaît :

**ANÉANTIR LE MONDE**

Ce bouton déclenche une **fin alternative du jeu**.

---

# 🧠 Astuce importante

Le jeu contient **plusieurs secrets dans l’interface**.

Un joueur curieux pourrait découvrir :

* des indices
* des raccourcis
* des accès cachés

Il est même possible de **terminer le jeu directement depuis l’interface** si vous trouvez la bonne méthode.
